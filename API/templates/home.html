Bienvenidos a 'The Loader'

Este es un pequeño programa que esta diseñado para leer grandes sets de data.

Permitira al usario cargar un archivo .xlsx para entender como funciona.

Este programa permite al usario crear, leer, actualizar o eliminar elementos en el archivo dado.

El siguiente link se utiliza para cargar el archivo a la base de datos.
